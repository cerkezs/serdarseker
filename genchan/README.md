# Gençhan Nakliyat — Yeni Nesil Kurumsal Web Sitesi

Statik HTML/CSS/JS site. Herhangi bir web sunucusunda (Apache, Nginx, Netlify, Vercel, GitHub Pages, paylaşımlı hosting) doğrudan yayınlanabilir.

## Yapı
```
index.html                 → Tek sayfa (tüm bölümler)
css/style.css              → Tüm stiller
js/main.js                 → Etkileşim + Three.js 3D parçacık ağı
js/three.min.js            → Three.js kütüphanesi (offline)
images/                    → Tüm görseller (logo, hero, ürün, harita, eski site görselleri)
genchan-nakliyat-gelistirme-raporu.pdf  → Detaylı analiz & yol haritası
```

## Özellikler
- 🎬 Sinematik hero (Three.js 3D parçacık ağı + sunset highway + 3D tır geçişi)
- 🎨 Modern kurumsal kimlik (lacivert + turuncu, Inter + Space Grotesk)
- 📱 Mobil öncelikli, fullscreen drawer menü
- ✨ Custom cursor, scroll-reveal, micro-interactions
- 🔍 SEO: title/meta/OG/JSON-LD MovingCompany schema
- ⚡ Preloader, lazy-loading, statik = hızlı
- 💬 WhatsApp floating + 3 ofis iletişim kartı
